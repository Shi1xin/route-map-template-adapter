# Template Parse Schema

Use this schema to describe a reference route-map image.

## Global Layout

- Canvas direction: vertical, horizontal, mixed.
- Top headers: main lanes or high-level categories.
- Side lanes: stages, methods, evidence, outputs.
- Center area: main content and route progression.
- Sections: dashed boxes, bands, grouped modules, repeated blocks.
- Title groups: a large section title and its small subtitle line belong to the same title group when they sit directly together.
- Narrow vertical boxes: stage labels and framework labels should use vertical stacked text.

## Node Types

- Stage: phase or task phase.
- Concept: key concept, variable, object, or construct.
- Rationale: foundation, analytical framework, mechanism claim.
- Content module: task, subtopic, analysis module.
- Method: document review, analysis method, model, case, interview, measurement.
- Evidence: data source, indicator, case, comparison, robustness check.
- Output: conclusion, path, recommendation, deliverable.
- Subtitle: short scope, dimension, or analytical focus attached to a title node.

## Connector Semantics

Assign a relation label to important connectors:

- supports
- derives
- measures
- drives
- tests
- compares
- feeds into
- forms
- translates into

## Quality Check

- The top headers describe the whole canvas.
- Side lanes align with central sections.
- Each central section has a stage role.
- Methods connect to the content they support.
- The main route has a visible progression from basis to analysis to mechanism to output.
